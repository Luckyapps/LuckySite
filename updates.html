  • UniFooter wurde hinzugefügt
    • Mit UniFooter (Universal Footer) werden die Links im Footer auf jeder Seite automatisch angepasst.
    • Das Festlegens des Footerinhalts wird in linkmanager.js vorgenommen. Dort kann in der initUniFooter() Funktion für jede Sprache ein eigener Footer eingestellt werden.
    • Über die Einstellung uniFooter kann die Funktion auf der gesammten Website deaktiviert werden.
    • Durch hinzufügen von <script>var blockUniFooter = true;</script> auf der HTML Seite wird die Funktion nur für diese Seite geblockt.
